# tech-training-camp-frontend

一个简单的markdown编辑器项目

## 一、项目的功能要求

### 必须实现的功能包括

- #### **完全支持GitHub的标准Markdown语法**（格式参考[这个文件](https://github.github.com/gfm/)）

- #### **支持使用顶部工具栏按钮进行操作**

- #### **左边输入Markdown内容，右侧实时显示渲染效果**

## 二、开发过程

### 基本目标

- [ ]  完成web界面的布局和样式

- [ ]  完成markdown语法转换

- [ ]  实现快捷工具栏

### 进阶目标

- [ ] 实现共享文档和协同编辑
