# tech-training-camp-frontend

一个简单的markdown编辑器项目

## 一、实现功能

- #### **支持GitHub的标准Markdown语法**（格式参考[Github规范](https://github.github.com/gfm/)）

- #### **支持使用顶部工具栏按钮进行操作**

- #### **左边输入Markdown内容，右侧实时显示渲染效果**
